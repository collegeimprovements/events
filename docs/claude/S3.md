# S3 API Reference

> **Always use `OmS3`** from `libs/om_s3` for S3 operations.

## Two API Styles

### Direct API (config as last argument)

```elixir
config = OmS3.config(access_key_id: "...", secret_access_key: "...")

# Basic operations
:ok = OmS3.put("s3://bucket/file.txt", "content", config)
{:ok, data} = OmS3.get("s3://bucket/file.txt", config)
:ok = OmS3.delete("s3://bucket/file.txt", config)
true = OmS3.exists?("s3://bucket/file.txt", config)

# Presigned URLs
{:ok, url} = OmS3.presign("s3://bucket/file.pdf", config)
{:ok, url} = OmS3.presign_put("s3://bucket/upload.jpg", config, expires_in: {5, :minutes})
```

### Pipeline API (chainable, config first)

```elixir
# Upload with metadata
OmS3.new(config)
|> OmS3.bucket("my-bucket")
|> OmS3.prefix("uploads/2024/")
|> OmS3.content_type("image/jpeg")
|> OmS3.metadata(%{user_id: "123"})
|> OmS3.put("photo.jpg", jpeg_data)

# From environment
OmS3.from_env()
|> OmS3.expires_in({5, :minutes})
|> OmS3.presign("s3://bucket/file.pdf")

# Batch with concurrency
OmS3.new(config)
|> OmS3.bucket("my-bucket")
|> OmS3.concurrency(10)
|> OmS3.put_all([{"a.txt", "content"}, {"b.txt", "content"}])
```

## S3 URIs

All operations accept `s3://bucket/key` URIs:

```elixir
"s3://my-bucket/path/to/file.txt"   # Full path
"s3://my-bucket/prefix/"             # For listing
"s3://my-bucket"                     # Bucket root
```

## Configuration

```elixir
# From environment
OmS3.from_env()

# Manual
OmS3.config(
  access_key_id: "AKIA...",
  secret_access_key: "...",
  region: "us-east-1"
)

# With proxy
OmS3.config(
  access_key_id: "...",
  secret_access_key: "...",
  proxy: {"proxy.example.com", 8080}
)

# LocalStack / MinIO
OmS3.config(
  access_key_id: "test",
  secret_access_key: "test",
  endpoint: "http://localhost:4566"
)
```

**Environment Variables:**

| Variable | Description |
|----------|-------------|
| `AWS_ACCESS_KEY_ID` | Required |
| `AWS_SECRET_ACCESS_KEY` | Required |
| `AWS_REGION` / `AWS_DEFAULT_REGION` | Default: "us-east-1" |
| `AWS_ENDPOINT_URL_S3` / `AWS_ENDPOINT` | Custom endpoint |
| `HTTP_PROXY` / `HTTPS_PROXY` | Proxy configuration |
| `S3_BUCKET` | Default bucket |

## Core Operations

| Function | Description |
|----------|-------------|
| `put/3-4` | Upload content |
| `get/2` | Download content |
| `delete/2` | Delete object |
| `exists?/2` | Check existence |
| `head/2` | Get metadata |
| `list/2-3` | List objects (paginated) |
| `list_all/3` | List all (handles pagination) |
| `copy/3` | Copy within S3 |
| `presign/2-3` | Generate presigned URL |
| `presign_get/2-3` | Presigned download URL |
| `presign_put/2-3` | Presigned upload URL |

## Batch Operations

All support glob patterns and parallel execution:

```elixir
# Upload multiple
OmS3.put_all([{"a.txt", "..."}, {"b.txt", "..."}], config, to: "s3://bucket/")

# Download with globs
OmS3.get_all(["s3://bucket/*.pdf"], config)

# Delete with patterns
OmS3.delete_all(["s3://bucket/temp/*.tmp"], config)

# Copy with glob
OmS3.copy_all("s3://source/*.jpg", config, to: "s3://dest/")

# Presign multiple
OmS3.presign_all(["s3://bucket/*.pdf"], config, expires_in: {1, :hour})
```

## File Name Normalization

```elixir
OmS3.normalize_key("User's Photo (1).jpg")
#=> "users-photo-1.jpg"

OmS3.normalize_key("report.pdf", prefix: "docs", timestamp: true)
#=> "docs/report-20240115-143022.pdf"

OmS3.normalize_key("file.txt", uuid: true)
#=> "file-a1b2c3d4-e5f6-7890-abcd-ef1234567890.txt"
```

## Pipeline Setters

| Setter | Description |
|--------|-------------|
| `bucket/2` | Set bucket name |
| `prefix/2` | Set key prefix |
| `content_type/2` | Set content type |
| `metadata/2` | Set metadata map |
| `acl/2` | Set ACL |
| `storage_class/2` | Set storage class |
| `expires_in/2` | Set presign expiry |
| `method/2` | Set HTTP method |
| `concurrency/2` | Set max concurrency |
| `timeout/2` | Set timeout |

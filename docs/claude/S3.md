# OmS3 API Reference

> **Always use `OmS3`** from `libs/om_s3` for S3 operations. Works with AWS S3, MinIO, RustFS, Cloudflare R2, DigitalOcean Spaces, Backblaze B2, Wasabi, LocalStack.

## Quick Start

```elixir
config = OmS3.config(access_key_id: "...", secret_access_key: "...", region: "us-east-1")

:ok = OmS3.put("s3://bucket/file.txt", "content", config)
{:ok, data} = OmS3.get("s3://bucket/file.txt", config)
:ok = OmS3.delete("s3://bucket/file.txt", config)
true = OmS3.exists?("s3://bucket/file.txt", config)
```

## API at a Glance

### Core Operations

| Function | Returns | Description |
|----------|---------|-------------|
| `put/3-4` | `:ok \| {:error, Error.t()}` | Upload content |
| `get/2` | `{:ok, binary} \| {:error, Error.t()}` | Download content |
| `delete/2` | `:ok \| {:error, Error.t()}` | Delete object (idempotent) |
| `exists?/2` | `boolean` | Check existence |
| `head/2` | `{:ok, map} \| {:error, Error.t()}` | Get metadata without download |
| `list/2-3` | `{:ok, %{files: list, next: token}} \| {:error, Error.t()}` | List objects (paginated) |
| `list_all/2-3` | `{:ok, [map]} \| {:error, Error.t()}` | List all (auto-paginate) |
| `copy/3` | `:ok \| {:error, Error.t()}` | Server-side copy |

### Presigned URLs

| Function | Returns | Description |
|----------|---------|-------------|
| `presign/2-3` | `{:ok, url} \| {:error, Error.t()}` | Presigned URL (GET or PUT) |
| `presign_get/2-3` | `{:ok, url} \| {:error, Error.t()}` | Presigned download URL |
| `presign_put/2-3` | `{:ok, %{url, fields}} \| {:error, Error.t()}` | Presigned upload form |

### Batch Operations (parallel, glob-aware)

| Function | Returns | Description |
|----------|---------|-------------|
| `put_all/2-3` | `[{:ok, uri} \| {:error, uri, reason}]` | Upload multiple |
| `get_all/2-3` | `[{:ok, uri, binary} \| {:error, uri, reason}]` | Download multiple |
| `delete_all/2-3` | `[{:ok, uri} \| {:error, uri, reason}]` | Delete multiple |
| `copy_all/2-3` | `[{:ok, src, dst} \| {:error, src, reason}]` | Copy multiple |
| `presign_all/2-3` | `[{:ok, uri, url} \| {:error, uri, reason}]` | Presign multiple |

### Streaming (Large Files)

| Function | Returns | Description |
|----------|---------|-------------|
| `Stream.download/2-3` | `Enumerable.t()` | Chunked download stream |
| `Stream.download_to_file/3-4` | `:ok \| {:error, term}` | Download directly to file |
| `Stream.download_with_callback/3-4` | `:ok \| {:error, term}` | Download with per-chunk callback |
| `Stream.upload/3-4` | `:ok \| {:error, term}` | Multipart upload from stream |
| `Stream.upload_file/3-4` | `:ok \| {:error, term}` | Multipart upload from file |

### Utilities

| Function | Returns | Description |
|----------|---------|-------------|
| `uri/2` | `String.t()` | Build `s3://bucket/key` URI |
| `parse_uri/1` | `{:ok, bucket, key} \| :error` | Parse URI |
| `normalize_key/1-2` | `String.t()` | Sanitize filename for S3 |
| `config/1` | `Config.t()` | Create config from options |
| `from_env/0` | `Request.t()` | Config from env vars |

## Two API Styles

### Direct API (config as last argument)

```elixir
config = OmS3.config(access_key_id: "...", secret_access_key: "...")

:ok = OmS3.put("s3://bucket/file.txt", "content", config)
{:ok, data} = OmS3.get("s3://bucket/file.txt", config)
:ok = OmS3.delete("s3://bucket/file.txt", config)
:ok = OmS3.copy("s3://bucket/old.txt", "s3://bucket/new.txt", config)
{:ok, url} = OmS3.presign("s3://bucket/file.pdf", config, expires_in: {1, :hour})
```

### Pipeline API (chainable, config first)

```elixir
OmS3.new(config)
|> OmS3.bucket("my-bucket")
|> OmS3.prefix("uploads/2024/")
|> OmS3.content_type("image/jpeg")
|> OmS3.metadata(%{user_id: "123"})
|> OmS3.acl("public-read")
|> OmS3.put("photo.jpg", jpeg_data)

OmS3.from_env()
|> OmS3.expires_in({5, :minutes})
|> OmS3.presign("s3://bucket/file.pdf")

OmS3.new(config)
|> OmS3.bucket("my-bucket")
|> OmS3.concurrency(10)
|> OmS3.timeout({2, :minutes})
|> OmS3.put_all([{"a.jpg", data1}, {"b.jpg", data2}])
```

### Pipeline Setters

| Setter | Type | Description |
|--------|------|-------------|
| `bucket/2` | `String.t()` | Default bucket name |
| `prefix/2` | `String.t()` | Key prefix (e.g., `"uploads/2024/"`) |
| `content_type/2` | `String.t()` | MIME type for uploads |
| `metadata/2` | `map()` | Custom metadata headers |
| `acl/2` | `String.t()` | `"private"`, `"public-read"`, etc. |
| `storage_class/2` | `String.t()` | `"STANDARD"`, `"GLACIER"`, etc. |
| `expires_in/2` | `integer \| {n, unit}` | Presign expiration |
| `method/2` | `:get \| :put` | Presign HTTP method |
| `concurrency/2` | `pos_integer()` | Batch parallelism (default: `schedulers * 2`) |
| `timeout/2` | `integer \| {n, unit}` | Operation timeout |

## Configuration

```elixir
# AWS
OmS3.config(access_key_id: "AKIA...", secret_access_key: "...", region: "us-east-1")

# S3-compatible (path_style auto-detected for custom endpoints)
OmS3.config(access_key_id: "...", secret_access_key: "...",
  endpoint: "http://localhost:9000", region: "us-east-1")

# With proxy
OmS3.config(access_key_id: "...", secret_access_key: "...",
  proxy: {"proxy.company.com", 8080}, proxy_auth: {"user", "pass"})

# From environment
OmS3.from_env()  # Returns Request struct for pipeline
```

### Config Options

| Option | Default | Description |
|--------|---------|-------------|
| `access_key_id` | *required* | Access key |
| `secret_access_key` | *required* | Secret key |
| `region` | `"us-east-1"` | AWS region or provider region |
| `endpoint` | `nil` | Custom endpoint (MinIO, R2, etc.) |
| `path_style` | auto | `true` when endpoint set, `false` for AWS |
| `connect_timeout` | `30_000` | TCP timeout (ms) |
| `receive_timeout` | `60_000` | Response timeout (ms) |
| `pool_timeout` | `5_000` | Pool checkout timeout (ms) |
| `max_retries` | `3` | Transient failure retries |
| `proxy` | `nil` | `{host, port}` or URL string |
| `proxy_auth` | `nil` | `{user, pass}` |
| `no_proxy` | `[]` | Bypass patterns |
| `transfer_acceleration` | `false` | AWS-only (raises for non-AWS) |

### Environment Variables (priority order)

| S3 Variable | AWS Fallback |
|-------------|-------------|
| `S3_ACCESS_KEY_ID` | `AWS_ACCESS_KEY_ID` |
| `S3_SECRET_ACCESS_KEY` | `AWS_SECRET_ACCESS_KEY` |
| `S3_REGION` | `AWS_REGION` / `AWS_DEFAULT_REGION` |
| `S3_ENDPOINT` | `AWS_ENDPOINT_URL_S3` / `AWS_ENDPOINT` |
| `HTTP_PROXY` | `HTTPS_PROXY` |
| `NO_PROXY` | - |

## Batch Operations

```elixir
# Upload multiple files
results = OmS3.put_all([{"a.txt", "..."}, {"b.txt", "..."}], config,
  to: "s3://bucket/uploads/", concurrency: 10)

# Download with globs
results = OmS3.get_all(["s3://bucket/docs/*.pdf", "s3://bucket/reports/*.csv"], config)

# Delete with globs
results = OmS3.delete_all(["s3://bucket/temp/*.tmp"], config)

# Copy with glob
results = OmS3.copy_all("s3://source/*.jpg", config, to: "s3://dest/images/")

# Presign multiple
results = OmS3.presign_all(["s3://bucket/*.pdf"], config, expires_in: {1, :hour})
```

### BatchResult Analysis

```elixir
alias OmS3.BatchResult

results = OmS3.put_all(files, config, to: "s3://bucket/")

# Quick checks
BatchResult.all_succeeded?(results)   #=> boolean
BatchResult.any_failed?(results)      #=> boolean

# Summary
summary = BatchResult.summarize(results)
# %{total: 100, succeeded: 95, failed: 5, success_rate: 0.95, ...}

# Categorize
BatchResult.failures(results)              #=> [%{uri: ..., reason: ..., error_type: ...}]
BatchResult.recoverable_failures(results)  #=> transient errors worth retrying
BatchResult.permanent_failures(results)    #=> access_denied, not_found, etc.
BatchResult.failures_by_type(results)      #=> %{not_found: [...], timeout: [...]}
BatchResult.failed_uris(results)           #=> ["s3://...", ...]

# Retry
BatchResult.retry_failures(results, fn uri, _reason ->
  OmS3.put(uri, get_content(uri), config)
end, max_attempts: 3, delay: 100, only_recoverable: true)

# Strict mode
BatchResult.raise_on_failure!(results)

# Format for logging
BatchResult.format(summary) #=> "Batch operation: 95/100 succeeded (95.0%), 5 failed [...]"
```

## Streaming

```elixir
alias OmS3.Stream, as: S3Stream

# Download to file
:ok = S3Stream.download_to_file("s3://bucket/large.zip", "/tmp/large.zip", config)

# Download as Elixir Stream
S3Stream.download("s3://bucket/large.zip", config, chunk_size: 10_485_760)
|> Stream.into(File.stream!("/tmp/output.zip"))
|> Stream.run()

# Download with callback
S3Stream.download_with_callback("s3://bucket/file.zip", config, fn chunk ->
  IO.binwrite(file, chunk)
end)

# Upload from file (multipart)
:ok = S3Stream.upload_file("/path/to/large.zip", "s3://bucket/large.zip", config)

# Upload from stream
File.stream!("/path/to/large.zip", [], 5_242_880)
|> S3Stream.upload("s3://bucket/large.zip", config,
  content_type: "application/zip", metadata: %{source: "upload"})
```

Raises `OmS3.StreamError` on failure (caught by `download_to_file`/`download_with_callback` rescue blocks).

```elixir
{:ok, url} = OmS3.presign_get("s3://bucket/file.pdf", config)
```

### Presign Cache

```elixir
# Supervision tree
{OmS3.PresignCache, name: MyApp.S3PresignCache}

{:ok, url} = OmS3.PresignCache.get_or_generate(
  MyApp.S3PresignCache, "s3://bucket/file.pdf", config, expires_in: {1, :hour})

OmS3.PresignCache.stats(MyApp.S3PresignCache)
```

## Error Handling

All operations return `{:error, %OmS3.Error{}}`:

```elixir
case OmS3.get("s3://bucket/file.txt", config) do
  {:ok, content} -> process(content)
  {:error, %OmS3.Error{type: :not_found}} -> :missing
  {:error, %OmS3.Error{type: :access_denied}} -> :forbidden
  {:error, %OmS3.Error{} = err} ->
    if FnTypes.Protocols.Recoverable.recoverable?(err), do: retry(), else: fail(err)
end
```

### Error Types

| Type | HTTP | Recoverable | Retry Strategy |
|------|------|-------------|----------------|
| `:not_found` | 404 | No | `:fail_fast` |
| `:access_denied` | 403 | No | `:fail_fast` |
| `:invalid_request` | 400 | No | `:fail_fast` |
| `:conflict` | 409 | No | `:fail_fast` |
| `:precondition_failed` | 412 | No | `:fail_fast` |
| `:request_timeout` | 408 | Yes | `:retry` (100ms base, 3 max) |
| `:connection_error` | - | Yes | `:retry` (100ms base, 3 max) |
| `:slow_down` | 503 | Yes | `:retry_with_backoff` (1s base, 5 max) |
| `:service_unavailable` | 503 | Yes | `:retry_with_backoff` (500ms base, 5 max) |
| `:internal_error` | 500 | Yes | `:retry_with_backoff` (500ms base, 3 max) |

### FnTypes Protocols

```elixir
# Recoverable protocol
FnTypes.Protocols.Recoverable.recoverable?(error)      #=> boolean
FnTypes.Protocols.Recoverable.strategy(error)           #=> :retry | :retry_with_backoff | :fail_fast
FnTypes.Protocols.Recoverable.retry_delay(error, attempt) #=> ms (exponential + jitter)
FnTypes.Protocols.Recoverable.max_attempts(error)       #=> integer
FnTypes.Protocols.Recoverable.severity(error)           #=> :transient | :degraded | :permanent | :critical
FnTypes.Protocols.Recoverable.trips_circuit?(error)     #=> boolean

# Normalizable protocol
FnTypes.Protocols.Normalizable.normalize(error)
#=> %FnTypes.Error{type: :not_found | :forbidden | :timeout | :external | ...}
```

## Network Layer

| Feature | Details |
|---------|---------|
| HTTP | Req + ReqS3 (AWS SigV4 signing) |
| Pooling | Finch (via Req), configurable `pool_timeout` |
| Retries | `retry: :safe_transient`, configurable `max_retries` (default 3) |
| Timeouts | `connect_timeout` (30s) / `receive_timeout` (60s) / `pool_timeout` (5s) |
| Proxy | HTTP/HTTPS with auth, `NO_PROXY` patterns, env var fallback |
| Errors | `OmS3.Error` structs with `Recoverable` protocol (backoff + jitter) |
| Telemetry | Automatic spans on all Client operations |
| S3-compat | Auto `path_style: true` for custom endpoints |

## Common Patterns

### Storage Module Wrapper

```elixir
defmodule MyApp.Storage do
  @bucket System.compile_env!(:my_app, :s3_bucket)
  defp config, do: OmS3.Config.from_env()

  def upload(path, content, opts \\ []) do
    OmS3.put("s3://#{@bucket}/#{path}", content, config(),
      content_type: Keyword.get(opts, :content_type, OmS3.ContentType.detect(path)),
      metadata: Keyword.get(opts, :metadata, %{}))
  end

  def download(path), do: OmS3.get("s3://#{@bucket}/#{path}", config())

  def url(path, opts \\ []) do
    OmS3.presign_get("s3://#{@bucket}/#{path}", config(),
      expires_in: Keyword.get(opts, :expires_in, {1, :hour}))
  end

  def delete(path), do: OmS3.delete("s3://#{@bucket}/#{path}", config())
  def list(prefix \\ ""), do: OmS3.list_all("s3://#{@bucket}/#{prefix}", config())
end
```

### Batch Upload with Retry

```elixir
results = OmS3.new(config)
|> OmS3.bucket("reports")
|> OmS3.prefix("monthly/")

  OmS3.BatchResult.retry_failures(results, fn uri, _reason ->
    content = regenerate_content(uri)
    OmS3.put(uri, content, config)
  end, max_attempts: 3)
end
```

### Error Handling in Practice

```elixir
# Simple pattern match
case OmS3.get(uri, config) do
  {:ok, content} -> {:ok, content}
  {:error, %OmS3.Error{type: :not_found}} -> {:error, :missing}
  {:error, %OmS3.Error{type: :access_denied}} -> {:error, :forbidden}
  {:error, %OmS3.Error{} = err} -> {:error, err.message}
end

# With recovery check
case OmS3.put(uri, data, config) do
  :ok -> :ok
  {:error, %OmS3.Error{} = err} ->
    if FnTypes.Protocols.Recoverable.recoverable?(err) do
      Process.sleep(FnTypes.Protocols.Recoverable.retry_delay(err, 1))
      OmS3.put(uri, data, config)  # single retry
    else
      {:error, err}
    end
end
```

### Cleanup Old Files
